风赢员工用API说明文档

2018.4.2 V1.0 by Y·H

概要说明

            API访问地址
              正式服：http://www.fnying.com/staff/api/

            返回数据格式
              json

            异常返回
              errcode 返回码
              errmsg  失败信息

              常见错误码：
              -1  系统繁忙，稍候再试
              110 服务异常中断，请与管理员联系
              120 参数错误

员工注册

http://www.fnying.com/staff/api/sign.php

            GET参数
              staff_name      员工姓名
              staff_phone     员工电话

            返回
              errcode = 0 请求成功

            说明
              风赢科技员工注册

员工情报一览

http://www.fnying.com/staff/api/staff_info_list.php

            参数
              无
              
            返回
              total     总记录件数
              rows      记录数组
                staff_id          员工ID
                staff_cd          员工工号
                staff_name        员工姓名
                nick_name         昵称（英文名）
                staff_avata       员工头像
                staff_sex         员工性别（0不明 1男 2女）
                staff_position    员工职位
                staff_mbti        员工性格（MBTI）
                staff_memo        员工个人简介
                staff_age         年龄（女性显示保密）
                staff_star_sign   星座
                online_status     在线状态（0签出 1签入）
                staff_subsidy     本周补助
                exp_balance       经费余额（分）
                join_date         加入时间
                
            说明

员工情报修改

http://www.fnying.com/staff/api/staff_info_edit.php

            GET参数
              staff_avata       员工头像
              nick_name         昵称（英文名）
              staff_memo        员工个人简介
              staff_mbti        员工性格
              
            返回
              errcode = 0     请求成功

            说明
              只有本人才能修改

任务一览

http://www.fnying.com/staff/api/task_list.php

            GET参数
              limit     （记录条数，可选）默认10 最大100
              offset    （记录偏移量，可选）默认0 与limit参数一起分页使用。如设置 offset=20&limit=10 取第21-30条记录
              
            返回
              total     总记录件数
              rows      记录数组
                task_id         任务ID
                task_name       任务
                task_intro      任务内容
                respo_name      责任人
                check_name      监督人
                task_level      任务等级
                task_value      任务价值
                task_perc       任务进度
                task_status     任务状态
                limit_time      任务期限
                ctime           创建时间
                
            说明
              本人相关任务一览

任务修改

http://www.fnying.com/staff/api/task_edit.php

            GET参数
              task_id         任务ID
              task_name       任务
              task_intro      任务内容
              respo_name      责任人
              check_name      监督人
              task_level      任务等级
              task_value      任务价值
              task_perc       任务进度
              task_status     任务状态
              limit_time      任务期限
              
            返回
              errcode = 0     请求成功

            说明
              权限不同，可修改的项目不同

            GET参数
              sign_type           签到类型

              longitude           经度

            返回
              errcode = 0     请求成功

            说明
              风赢科技员工签到

办公室签到记录

http://www.fnying.com/staff/api/office_sign_log.php

            Get参数
              limit     （记录条数，可选）默认10 最大100
              offset    （记录偏移量，可选）默认0 与limit参数一起分页使用。如设置 offset=20&limit=10 取第21-30条记录

            返回
              total     总记录件数
              rows      记录数组
                log_id      签到日志ID
                staff_name  员工姓名
                sign_type   签到类型
                ctime       签到时间
