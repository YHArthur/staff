风赢员工用API说明文档

2018.4.2 V1.0 by Y·H

概要说明

            API访问地址
              正式服：http://www.fnying.com/staff/api/

            返回数据格式
              json

            异常返回
              errcode 返回码
              errmsg  失败信息

              常见错误码：
              -1  系统繁忙，稍候再试
              110 服务异常中断，请与管理员联系
              120 参数错误

员工注册

http://www.fnying.com/staff/api/sign.php

            GET参数
              staff_name      员工姓名
              staff_phone     员工电话

            返回
              errcode = 0 请求成功

            说明
              风赢科技员工注册

员工情报一览

http://www.fnying.com/staff/api/staff_main_all.php

            GET参数
              limit     （记录条数，可选）默认10 最大100
              offset    （记录偏移量，可选）默认0 与limit参数一起分页使用。如设置 offset=20&limit=10 取第21-30条记录
              
            返回
              total     总记录件数
              rows      记录数组
                staff_id          员工ID
                staff_cd          员工工号
                staff_name        员工姓名
                staff_avata       员工头像
                staff_sex         员工性别（1男 2女）
                staff_mbti        员工性格
                staff_memo        员工个人简介
                staff_age         年龄（女性显示保密）
                staff_star_sign   星座
                join_date         加入时间

            说明

办公室签到

http://www.fnying.com/staff/api/office_sign.php

            GET参数
              sign_type           签到类型
              latitude            纬度
              longitude           经度

            返回
              errcode = 0     请求成功

            说明
              风赢科技员工签到

办公室签到记录

http://www.fnying.com/staff/api/office_sign_log.php

            Get参数
              limit     （记录条数，可选）默认10 最大100
              offset    （记录偏移量，可选）默认0 与limit参数一起分页使用。如设置 offset=20&limit=10 取第21-30条记录

            返回
              total     总记录件数
              rows      记录数组
                log_id      签到日志ID
                staff_name  员工姓名
                sign_type   签到类型
                ctime       签到时间
